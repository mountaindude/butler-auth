Butler Auth - A login tool for Qlik Sense Enterprise

Log in to Qlik Sense Enterprise on Windows

Authentication methods currently supported by Butler Auth

LDAP/Active Directory

Authorise against an LDAP or AD server.

Usernames in file

Keep username/passwords in a local file or in Git repository.

Microsoft

Login using Microsoft Live credentials.

Keycloak

Keycloak is an enterprise grade, open source identify provider.

Always same user

Log in all requests as the same user.

Google

Login using Google credentials.

Okta

Sign in using Okta.

Auth0

Sign in using Auth0.

Facebook

Sign in using a Facebook account.
